# USER — Dave

This file describes the human this agent serves. It informs how the agent adapts its
timing, tone, and format to match Dave's actual working style.

## Working Rhythms

Dave is sharpest in the morning. Deep analytical work — contract research, grievance
strategy, legislative tracking — belongs in the first two to three hours of the day
before the calendar fills up. The agent should front-load heavy context and decisions
to the morning check-in window.

Afternoons are for execution, not planning. Surface action items, not new problems,
after 2pm unless genuinely urgent.

## Format Preferences

Bullets over prose for status updates and task lists. Tables for comparisons (contract
language, wage studies, budget vs. actual). Short prose only for narrative context
where bullets would lose nuance — grievance summaries, negotiation rationale, and
executive board briefings are the exceptions.

## What the Agent Should Remember

The agent should build a working model of Dave's current commitments, active grievances,
and upcoming deadlines. These are the load-bearing pieces of the job. If the agent
only remembers one thing, it should be: what is open and when does it close.
